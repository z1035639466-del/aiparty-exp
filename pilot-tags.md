## 二、净收益逐项勾选依据（G1–G5 / L1–L3，工具箱@2026-07）

判法（尺 v2.1）：语义净额——增值 ≥1 且无显著减值 → 增值；互抵或皆无 → 中性；减值主导 → 减值。
未列出的通道均为 ✗（无实质贡献）。凡"含混通道硬凑"风险处已明示放弃勾选。

### 试点主体 8 条

**Beer Pong — 中性（互抵）**
- G1 ✗ 进杯与否人人眼见，系统无公正增益；G2 ✗ 无隐藏信息；G3 ✓ 剩杯数/回合归属/重掷权的留痕可卸载（2v2 长局中"该谁投、还剩几杯"确有记账负担）；G4 ✗ 轮掷本身即节拍；G5 ✗ 语料未标风险，不硬凑。
- L1 ✓ 进杯是全场眼见为实的高光瞬间，系统旁录削弱不了也替代不了；L2 ✓ 每掷后录入即打断投掷手感节奏；L3 ✗ 录入频率随回合尚可接受。
- 净额：G3 与 L1/L2 互抵 → **中性**。

**Quarters — 减值（减值主导）**
- G1 ✗ 弹中与否眼见；G2 ✗；G3 ✗ 弹中即指定饮、无跨局状态可卸载；G4 ✗；G5 ✗ 语料风险以 S 位承接（见下），不重复计入收益通道以免硬凑。

## 三、A 段对照 8 条 · 选择理由

**"标注强在哪"型（旧 A 注明确主张手机化的机制性优势）：**

**"零损耗/可行性"型（旧 A 注只主张映射可行，未主张体验增益）：**
5. **Yo Nunca**——A 注"是六原语的教科书级映射"，可行性表述的原型句；用于检验零损耗型是否也可能复核出实益（结果：增值）。
