- **B4 实体动作/道具增益**：实体牌、纸笔、肢体动作本身成为笑点或仪式（蒸自"把'蟑螂'拍到

**减值通道（去数字化损失，蒸自损失侧 · 中立源）**
- **L'1 实时/物理本体不可替**：手速、投掷、物理引擎乐趣无点击替代（蒸自 C 级全组："实时
  物理弹跳/碰撞""真实投掷躲避的肢体反应速度"）。
- **L'2 空间/目击信息损失**：地图移动、视觉目击、场景沉浸被砍导致玩法素材贫化（蒸自
  "砍掉地图跑动与视觉目击""砍AR扫描与3D跑图"）。
- **L'3 自由输入/自动化功能损失**：数字画布、打字匿名提交、自动去重/拼图等系统功能被砍
  （蒸自"砍掉手机数字画布""砍手机打字输入""砍自动拼图生成"）。
- **L'4 精度/连续性损失**：连续量离散化、判定精度钝化（蒸自"连续滑块砍成粗粒度离散按钮,
  损失精细度"）。

## 3. 逐条摘录 · 选择理由 · 通道勾选依据（每勾附落点句）

### 3.1 A 级拟入弹药 3 条

**Liar's Bar — 增值**
- 原评级摘录：**A**｜原评语摘录："出牌自称点数可被质疑,输家俄罗斯轮盘赌｜诈唬+命运仪式｜
  random定弹膛+state记存活+show仅本人见手牌｜**真人扣扳机前的沉默压强远超任何屏幕音效**"。
- 选择理由：近两年 Steam 派对爆款中市场验证最强者，弹药池头部候选——排期尺先测头部；
  诈唬＋命运仪式双家族复合；其评语是 B2 通道最纯的原文样本。
- B1 ✓ 落点：出牌"自称点数"即口头诈唬，质疑判定落 ask（是否质疑）＋ show 翻牌验真——
  读脸决定是否质疑，微表情即博弈输入。
- B2 ✓ 落点：轮盘环节 random 定弹膛后结果由 fx 揭晓，扣扳机动作保留给真人——全场屏息
  的"沉默压强"（原文直书）落在揭晓瞬间。
- B3 ✗ 口头自称已计入 B1 的诈唬输入，不重复计；B4 ✗ 可配实体牌但非本条命门，不硬凑。
- L'1–L'4 ✗ 原评"A=原生映射，无需阉割"，机制无砍割；数字版 3D 酒馆美术属表皮非机制损失。
- 净额：B1/B2 无显著减值 → **增值**。

**Coup — 增值**
- 原评级摘录：**A**｜原评语摘录："隐藏身份对应特权,可诈称身份行动也可被挑战｜诈唬+信息
  不对称｜random发身份+show身份仅自己可见+ask选择行动/是否质疑｜**面对面观察微表情猜身份
  是纯线上版做不到的核心资产**"。
- 选择理由：候选池型（家族 8）在弹药池中的代表；原语映射全表最干净（random 发＋show 私见＋
  ask 质疑），且与既有生成词汇"声明质疑"槽直接衔接（C7 文法 `prop_reveal:身份牌` 先例）；
  评语是 B1 通道最纯原文样本。
- B1 ✓ 落点：诈称行动与"是否挑战"决策环节——ask 质疑前的读脸即核心博弈输入（原文直书
  "核心资产"）。
- B3 ✓ 落点：行动宣告保口头（"我用公爵收税"），系统只在被质疑时 prop_reveal 亮身份牌——
  撒谎自证的临场表演张力落在宣告瞬间。
- B2 ✗ 有翻牌验真但原文未主张全场仪式，不硬凑；B4 ✗。
- L'1–L'4 ✗ 纯卡牌逻辑，A 级原生映射无砍割。
- 净额：B1/B3 无显著减值 → **增值**。

**Codenames — 增值**
- 原评级摘录：**A**｜原评语摘录："领袖给一词一数提示,队友靠联想猜词卡｜竞猜+联想推理｜
  show棋盘(身份仅领袖可见)+ask猜词点击｜**提示词由真人口头说出,全场紧张听线索的氛围无法
  被打字替代**"。
- 选择理由：受限沟通系代表（DM-skill §8 结果机制 13"受限沟通：一词一数/是否无关三态"
  点名 Codenames），正踩采集库合作系结构性盲区；"提示保口头"是 dir:A 数7 案
  "保本体动作者可增值"判语在 B 向的对应试金石。
- **家族挂靠核验（提名 42 依据）**：36 表内受限沟通仅两支——"是否/无关三态"支
  （29 情景谜题问答链、33 二十问排除，均为概念空间边界件）与表演支（9 表演猜词，
  非语言传词）——皆不覆盖"一词一数"口头受限提示；扩编 38「集体去重提示」为 Just One 型
  全员同时写提示去重，与单领袖一词一数不同构；37/39/40/41 号已被三裁占用。
  36＋扩编皆无对应 → **正式提名第 42 家族「一词一数受限提示竞猜」**，待人工照准
  （提名已入 BACKLOG）。三问树预归：Q1 否（谜底为词卡非在场人）→ Q2 否 → Q3 是 →
  广播型（叠分队修饰符）。
- B3 ✓ 落点：提示环节保真人口头（一词一数），系统只 show 双层棋盘（身份图仅领袖私见）——
  本体动作（口头给词）未被替换，正合 04 §3 判语。
- B2 ✓ 落点：猜词 ask 点击后 show 全场同步翻色——全桌同时看到翻对/翻错（含刺客）瞬间的
  哗然；原文"全场紧张听线索的氛围"落在同步聆听与揭晓。
- B1 ✗ 有观察领袖表情的成分但原文未主张，不硬凑；B4 ✗ 实体词卡可用但非命门。
- L'1–L'4 ✗ 打字提示本非该游戏线下乐趣，砍之无损；A 级原生映射。
- 净额：B3/B2 无显著减值 → **增值**。

### 3.2 B 级 3 条（原评语均为"砍掉什么保留什么"型）

**Among Us — 中性**
- 原评级摘录：**B**｜原评语摘录（砍/保型）："**砍掉地图跑动与视觉目击,任务简化成手机点击
  清单,讨论投票环节完整保留**｜真人围坐讨论时读微表情比线上文字聊天更强"。
- 选择理由：全表最重的一刀——砍掉的"视觉目击"不是皮而是指控的证据来源；本条是尺样异源
  原则要求的关键样本："线下增强点"列（仪器出生地）说它强，损失侧说它伤筋动骨——
  正是"仪器出生地预测其会死"的条目。
- B1 ✓ 落点：紧急会议讨论投票环节——围坐读微表情（原文直书"比线上文字聊天更强"），
  投票落 ask。
- B2/B3/B4 ✗ 原文未主张其他增益，不硬凑。
- L'2 ✓ 落点：地图跑动与视觉目击被砍——线上版"我看见他从尸体旁走开"类目击证词是指控的
  素材来源，砍后讨论素材退化为点击记录；任务玩法空心化为清单点击；内鬼破坏（sabotage）
  的张力同遭砍除。损失显著。
- L'1 ✗（跑动非手速乐趣本体）；L'3 ✗；L'4 ✗。
- 净额：B1 实在但 L'2 显著——讨论被增强的同时讨论的素材被抽薄 → 互抵 → **中性**。

**Gartic Phone — 增值**
- 原评级摘录：**B**｜原评语摘录（砍/保型）："**砍掉手机数字画布,改用纸笔实体传阅,手机只发
  主题词/计时/收尾轮流展示投票**｜纸笔传递时忍笑憋不住的连锁反应是线下独有名场面"。
- 选择理由：三条 B 里唯一"砍掉数字件反而换来实体名场面"的方案——测尺能否区分"砍掉=降级"
  与"砍掉=换更好的实体件"（L'3 与 B4 同源对撞的样本）。
