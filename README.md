# ZAKZOK(代号 Yappa)· AI 活局长

> **北极星:这是线下社交游戏局的活局长,不是聊天局,不是反应手感游戏 App。**
> 游戏发生在桌上、身体上、道具上、人和人的眼神里——不发生在屏幕里。
> 文字是组织手段,不是游戏场地。

任何人(和任何 agent)动手前先读宪法全文:
**[docs/records/裁定-产品定位-线下社交游戏局-20260722.md](docs/records/裁定-产品定位-线下社交游戏局-20260722.md)**

## 手机的四个角色(仅此四个)

1. **入座票**:进桌、看进度、三信号
2. **私密信道**:藏信息的唯一载体——牌卡、私骰、技能牌、额头牌
3. **判定仪器**:显式判定时刻才抬起来——拔枪、开牌、拍照/录音判定
4. **局长的喇叭**:台词与音乐

## 动手前的判据(全文见宪法,这里是门口的牌子)

- 新功能让玩家**盯屏时间变长**而不是让桌上的人**互相动起来**?默认拒。
- **道具只发不代玩**:结果由玩家玩出来;替玩家摇骰/翻牌/出结果 = 与虚构同罪。
- 私发=发牌不是私聊;藏信息长在道具上,不长在对话里。
- 道具技术选型走**五步判据**(拆时刻→问感官通道→通道决定技术→基线先做全→引擎线永久关闭)。
