# Tier-2 公开 UGC 河流发现：越南、泰国、印尼、波兰、尼日利亚

日期：2026-07-16
对应机器证据：[tier2_public_evidence_2026-07-16.json](./tier2_public_evidence_2026-07-16.json)

## 先说结论

本轮完成的是《UGC 河流采集地图》中 P3 Tier-2 的**公开网页补充扫面**，覆盖越南、泰国、印度尼西亚、波兰和尼日利亚，共整理 60 条分级证据。它回答了三个问题：

1. 当地有哪些接近“局头”的角色词，但角色到底是“管规则”“组织场子”还是“负责气氛”；
2. 当地年轻人围绕聚会游戏、喝酒游戏和朋友局使用哪些本地检索词、标题套路和口语；
3. 哪些玩法已经可以拆成安全的机制原子，哪些只能作为风险语料观察。

最重要的发现不是找到五个一一对应的译词，而是看到各地都存在角色分裂：

| 地区 | 最接近“规则/流程负责人” | 最接近“气氛负责人” | 判断 |
|---|---|---|---|
| 越南 | **chủ xị**（南部酒局中的主持/控制角色）、**quản trò / MC dẫn dắt**（通用带游戏者） | 暂未发现稳定、专属的年轻人口语称呼 | chủ xị 很强，但有地域和饮酒语境限制 |
| 泰国 | **คนเปิดกฎ / อ่านกฎ**（开规则、读规则的人，只是临时功能） | **สายปาร์ตี้ / สายดริ๊งก์ / สายเมา / สายเฮฮา** 是玩家身份，不是主持角色 | 尚无足够证据证明存在稳定的“局头”单词 |
| 印尼 | **MC / pemandu permainan**（专业主持/游戏带领者） | **badut di tongkrongan / pencair suasana**（朋友局里的搞笑、破冰、提气氛者） | “管流程”和“搞气氛”明显是两类人 |
| 波兰 | **Mistrz Gry**（游戏主持/规则控制者） | **dusza towarzystwa**（派对灵魂人物）；**wodzirej** 更偏职业带场 | Mistrz Gry 是本轮最接近“局头手册”内容入口的角色词 |
| 尼日利亚 | **host / MC**（管流程、节目和游戏） | **hypeman / hype-person**（设定情绪、调动人群、与 DJ 配合） | 角色分工最清楚，适合双角色采集 |

因此，后续不能只把“局头”翻译成 host 再搜。更有效的方式是把角色拆成三条河并行搜：

- **规则河**：game master、rule reader、dealer、MC、pemandu permainan；
- **组织河**：host、chủ xị、party planner、event host；
- **气氛河**：hypeman、pencair suasana、dusza towarzystwa，以及各地“派对咖/酒局咖”玩家身份词。

本报告只写入能够从公开网页复核的内容，没有绕过登录、没有读取私密内容、没有点赞评论或关注账号。

需要特别说明：

- TikTok 公开搜索页在本轮环境中被 robots 限制，不能把本轮说成“已经看完五国 TikTok 信息流和评论区”。
- 越南有可直接复核的 **TikTok Shop Vietnam** 商品页、标签和公开评价文本，因此越南的站内商品/评论证据最强。
- 尼日利亚有本地媒体明确记录 TikTok 规则系列、病毒音频和 hypeman 内容流；这是“媒体对 TikTok 内容的可复核报道”，不是本轮直接播放过原视频。

- App Store、Google Play 的国家页面适合发现本地查询词和产品机制；发布者可能自行本地化，不能单凭一页就断言某个词已经是当地自然黑话。

置信度规则：

- **高**：当地来源直接把本地词、角色或玩法放进聚会/酒局语境；
- **中**：本地商店页、商业内容、二手报道或职业场景，仍需站内密度验证；
- **低**：只保留用于防止误判的一词多义线索。本轮结构化证据没有用低置信度条目填数量。

## 1. 越南

### 角色词

**chủ xị** 是本轮越南最重要的词。[《Báo Cần Thơ》关于南部酒文化的文章](https://baocantho.com.vn/ban-them-ve-van-hoa-ruou-o-nam-bo-a21213.html)把它放在南部饮酒圈里：一只共用杯由 chủ xị 管理。它有“控制节奏/轮次”的味道。[《Báo Bình Định》的语言文章](https://baobinhdinh.vn/viewer.aspx?mabb=146944&macm=6&macmp=7)又把常见意义解释到“一场酒局的主人或主事者”。

这使它非常接近“局头”，但不能过度外推：

- 它与饮酒场景绑定；
- 第一条证据明显偏越南南部；
- 它可能同时含有主办、主持、出资或掌控轮次等历史义项，不等于所有年轻人都把游戏主持叫这个词。

**quản trò / MC dẫn dắt** 是更通用、更职业化的带游戏角色。[本地聚会游戏文章](https://www.mocquan.vn/blogs/news/dung-ngoi-khong-yen-voi-top-5-tro-choi-tiec-van-minh-hap-dan)说 quản trò 或 MC 能带领活动、拉高气氛；[nShop 的聚会玩法](https://www.nshop.com.vn/blogs/go-hobby-news/choi-gi-trong-tiec-don-giao-thua)则让 quản trò 倒数后由玩家猜歌。它更像“主持人/游戏带领者”，不如 chủ xị 口语化。

### 商品和评论里长出来的检索词

越南 TikTok Shop 的价值很高，因为商品标题、标签和评价会自然形成查询簇：

- **bài Huệ / NocOut / KOSAY**：[68 张挑战牌商品](https://shop-vn.tiktok.com/pdp/1730043106653407720)；
- **cực lầy / nặng đô / lá liên kết**：[带公开评价的 bài Huệ 商品](https://shop-vn.tiktok.com/pdp/1729496541567879214)；
- **U Là Trời / bài uống rượu / game tiệc / ăn nhậu / game say rượu**：[50 张 18+ 商品](https://shop-vn.tiktok.com/pdp/1730395889607149668)；
- **gamenhau / bainhau / trò chơi uống rượu**：[Trải Lòng For Couple](https://shop-vn.tiktok.com/pdp/1731165167699133813)；
- **Uống Đê**：[83 张朋友聚会牌](https://shop-vn.tiktok.com/pdp/1729597225663892283)；
- **XÍCH**：[情侣 69 张卡牌](https://shop-vn.tiktok.com/pdp/1729773072180153245)，属于成人亲密问答支流。

这些词不是同一类：

- bài Huệ、NocOut、U Là Trời、Uống Đê、XÍCH 更像产品名或内容 IP；
- bài uống rượu、game nhậu、bainhau、gamenhau 是品类入口；
- cực lầy、nặng đô 是强度/风格描述；
- lá liên kết 是规则内的卡牌类型。

### 可拆机制

最有新鲜度的是公开评价里出现的“两手机玩法”：

1. 一部手机录入所有玩家名字并转轮盘；
2. 另一部手机随机抽牌；
3. 被选中者执行卡牌任务；
4. 某些任务延迟到两轮后触发；
5. lá liên kết 可以叠加或翻倍结果。

这里可安全吸收的是“双端分工、延迟触发、联结卡”三个机制。不能吸收的是“喝两三杯、喝完、翻倍罚酒”等高强度结果。

### 下一轮优先查询

chủ xị bàn nhậu；chủ xị trò chơi；quản trò trò chơi tiệc；bài Huệ；NocOut；KOSAY；bài uống rượu；game nhậu；gamenhau；bainhau；Uống Đê；cực lầy；nặng đô；lá liên kết。

## 2. 泰国

### 没有找到可以诚实等号的“局头词”

泰国本轮最重要的结论是：**不要强行把玩家身份词当主持角色**。

- **สายปาร์ตี้**：派对型玩家；
- **สายดริ๊งก์**：饮酒型玩家；
- **สายเมา**：偏醉饮/喝酒的玩家；
- **สายเฮฮา**：热闹搞笑型玩家；
- **สายแข็ง**：酒量强、强度高的玩家。

这些词适合找到内容和人群，不表示“谁建局、谁管规则”。

[Lemon8 的五种 เกมวงเหล้า 帖子](https://www.lemon8-app.com/%40nightlife.run/7476034298936820241?region=th)只给出了临时功能：有人 **เปิดกฎ / อ่านกฎ**，即开规则或读规则；争议出现时读给大家听。这可以作为“规则人”查询种子，但证据不足以把它包装成固定头衔。

### 泰国 UGC 的安全设计反而很成熟

同一篇 Lemon8 UGC 提供了一套可直接进入产品规范的安全设计：

- 开始前共同确认三条规则；
- 可以暂停；
- 先约定 no-go zone；
- 不强迫触摸或给别人打电话；
- 不想做可以 pass，换成讲故事或其他动作；
- 两三人也能改规则玩；
- 可把任务放在 Notes，用网页/手机随机数，再投到电视或平板。

[Spin The Shot 帖子](https://www.lemon8-app.com/%40mai.in.th/7518685060525507090?region=th)又把转盘拆成“喝、回答、做任务”，并明确给出无酒精饮料、水、食物和限量方案。这类内容比单纯的“罚酒”更值得采。

### 玩法词和机制

- **เกมวงเหล้า**：最强品类词；
- **เกมสายเมา / เกมไพ่วงเหล้า / เกมปาร์ตี้**：相邻查询簇；
- **ภารกิจ**：任务；
- **บทลงโทษ**：惩罚/后果，应与 consent、pass、ไม่ดื่มแอลกอฮอล์ 等安全词一起搜；
- 禁词、接龙故事、记忆链、两真一假、二十问、rapid fire：[十种聚会游戏 UGC](https://www.lemon8-app.com/%40my__rose/7546145743448490497?region=th)；
- 手指随机选人：[Finger Dice 商店页](https://play.google.com/store/apps/details?hl=th&id=com.ToolsCompany.FingerPicker)；
- 朋友照片格子加奖励/惩罚：[LuckyPoke](https://apps.apple.com/th/app/luckypoke-%E0%B9%80%E0%B8%81%E0%B8%A1%E0%B8%88-%E0%B8%A1%E0%B8%81%E0%B8%A5-%E0%B8%AD%E0%B8%87/id6757243641?l=th)；
- 自定义带权轮盘：[Lucky Wheel](https://apps.apple.com/th/app/lucky-wheel-%E0%B8%A7%E0%B8%87%E0%B8%A5-%E0%B8%AD%E0%B8%AB%E0%B8%A1-%E0%B8%99-chooser/id6752469323?l=th)。

[饮酒指令积木帖子](https://www.lemon8-app.com/%40kanzensouth2/7294556891479360002?region=th)里的 ดื่มรอบวง、หมดแก้ว 以及 สายแข็ง 是高风险观察词：适合做过滤器，不适合直接变成玩法。

### 下一轮优先查询

เกมวงเหล้า；เกมสายเมา；เกมไพ่วงเหล้า；สายปาร์ตี้；สายดริ๊งก์；สายเฮฮา；สายแข็ง；บทลงโทษ；ภารกิจ；คนเปิดกฎ；อ่านกฎ；ผ่านได้；ไม่บังคับ；ไม่ดื่มแอลกอฮอล์。

## 3. 印度尼西亚

### 最接近的是两套角色，而不是一个词

[Froyonion 的本地青年文章](https://www.froyonion.com/news/potensi-diri/alasan-kenapa-nongkrong-bisa-memberimu-banyak-inspirasi-baru)写到：

- **badut di tongkrongan**：朋友局里的“搞笑担当”；
- **pencair suasana**：破冰、让气氛松动的人；
- **mood booster**：拉高情绪的人。

这组词是“气氛角色”，不保证他负责建局、读规则或判定输赢。与之相对：

- **MC**：管正式流程、带 ice breaking 和 games；
- **pemandu permainan**：游戏带领者。

[本地 MC 服务文章](https://www.expedition17.com/2026/01/jasa-mc-outbound-bandung-bikin-acara-family-gathering-makin-seru.html)和[本地游戏服务页](https://tripee.co.id/product/games-indoor-bali)支持功能含义，但它们偏职业活动，不是 Gen-Z 自称。

### 核心语境不是 party，而是 nongkrong

印尼最值得放大的根词是：

- **nongkrong**：一起待着、聚着；
- **tongkrongan**：朋友局/常聚的圈子或场景；
- **sirkel**：圈子；
- **bestie**：好友；
- **aib**：糗事、隐私、黑历史类内容入口；
- **ngakak / pecah / membagongkan**：搞笑、炸场、离谱风格入口。

[Jawa Pos 的本地玩法文章](https://www.jawapos.com/hobi-kesenangan/016514020/10-permainan-seru-anti-boring-saat-nongkrong-bersama-teman-dan-keluarga)把 jujur atau tantangan 和转瓶选人放进 nongkrong；[Hipwee](https://www.hipwee.com/hiburan/6-ide-tantangan-gokil-buat-kongkow-bareng-teman-teman-berani-coba/)提供 Teruskan Lagu，也就是继续唱/接歌的轮次玩法。

### 可拆机制

- **jujur atau tantangan**：真心话或挑战；
- **turunkan jari jika**：符合条件就放下一根手指；
- **siapa yang paling mungkin**：谁最可能；
- **game impostor / pembohong**：隐藏角色、讨论、投票、最终猜词；
- **oper ponsel**：传手机；
- **tebak tebakan**：猜词/你演我猜；
- **Paranoia**：随机指定人再问大胆问题，属于成人高风险支流。

[Liar Game 商店页](https://play.google.com/store/apps/details?hl=id&id=io.jkgwak.liargame)的“隐藏骗子—传手机—讨论—投票—被指认者最后猜一次”是可以安全改造的短局模板。[Paranoia](https://play.google.com/store/apps/details?hl=id&id=com.fliportell)则必须放进成人、同意和跳过控制里。

### 下一轮优先查询

game buat nongkrong；tongkrongan；nongkrong bareng teman；badut tongkrongan；pencair suasana；jujur atau tantangan；siapa yang paling mungkin；turunkan jari jika；game impostor；oper ponsel；aib；membagongkan；ngakak；pecah。

## 4. 波兰

### Mistrz Gry 是本轮最强的规则角色

[LocoGames 波兰语商店页](https://play.google.com/store/apps/details?hl=pl&id=com.pmapplications.locogames)明确提供 **Klasyczny (tryb Mistrza Gry)**：Mistrz Gry 抽出任务、规则和投票。这个角色同时满足：

- 有人掌握卡牌/题库；
- 有人发布新规则；
- 有人推进轮次；
- 内容天然可以写成“主持人手册、备忘、规则笔记”。

因此波兰后续应优先搜：

- Mistrz Gry + domówka；
- Mistrz Gry + gry alkoholowe；
- zasady Mistrza Gry；
- pytania dla Mistrza Gry；
- wyzwania / kara + ekipa。

### 三个相似但不能混用的角色

- **Mistrz Gry**：游戏内规则控制者，最接近目标；
- **wodzirej**：职业活动带场者，会带比赛、运动和舞蹈游戏，[本地活动页](https://pca.rzeszow.pl/pozytywne-trio/)支持这一点；
- **dusza towarzystwa**：派对灵魂人物、社交中心，[TVP 的游戏文章](https://pytanienasniadanie.tvp.pl/37595222/gry-dzieki-ktorym-zostaniesz-dusza-towarzystwa)把游戏与这一身份关联，但它不表示掌控规则。

### 波兰聚会词根

[Party Bus Kraków 的派对词表](https://partybuskrakow.pl/slownik-imprezowy-a-z/)给出：

- **bifor / beforek**：正式派对前的预热；
- **bibka**：口语派对；
- **balety**：出去玩/派对；
- **domówka**：家中派对；
- **melanż**：派对/混局；
- **ekipa / ziomki**：一队朋友/朋友们。

本地开发的 [Melo](https://apps.apple.com/pl/app/melo-gra-imprezowa/id6451415571?l=pl)把 domówka、melanż、ekipa 与 Prawda czy Wyzwanie、Nigdy w życiu、问题和挑战牌组合起来，而且明确允许玩家无理由跳过任何题。这一点应成为所有成人聚会题库的默认交互。

### 可拆机制

- Mistrz Gry 抽任务、规则、投票；
- 自定义 Prawda czy Wyzwanie；
- 瓶子随机选人；
- 隐藏 traitor、讨论、投票、挑战；
- 开始前共同决定 kara；
- 记忆链与重复链；
- app-as-host 按节奏持续发题。

[Blok Ekipa. Na Melanżu](https://www.ceneo.pl/44425906)能验证 melanż、ziomki 和记忆重复链，但“不会/不做就喝”属于压力型饮酒规则，只能留作风险词。

### 下一轮优先查询

gry alkoholowe；gry na domówkę；domówka；melanż；bifor；beforek；bibka；ekipa；ziomki；Mistrz Gry；Mistrz Pytań；dusza towarzystwa；wodzirej；Prawda czy Wyzwanie；Nigdy w życiu；kara；wyzwania。

## 5. 尼日利亚

### host / MC 与 hypeman 的双角色最清晰

尼日利亚证据里，最接近“局头”的不是一个人，而是一套配合：

- **host / MC**：控制流程、节目、游戏和观众注意力；
- **hypeman / hype-person**：设定情绪、喊麦、call-and-response、把人群能量推高，常与 DJ 配合。

[Guardian Nigeria 对 hype culture 的解释](https://guardian.ng/life/culture-lifestyle/hype-culture-in-nigeria/)把 hypeman 写成设定 tone 和 mood、让人群参与的人。[OkayAfrica 2026 年的女性 hypeman 采访](https://www.okayafrica.com/what-its-like-to-be-a-female-hypeman-in-nigeria/1427020)进一步称其为“energy conductor”，带着派对穿过 DJ 的不同段落。

[Punch 的采访](https://punchng.com/my-life-as-bad-boy-timz-hypeman-sceed-barms/)给出了最清楚的角色切换：event host 在正式流程中协调节目，到了 after-party 又可以变成 life of the party 的 hypeman。[2026 年校园巡演报道](https://punchng.com/dj-dewik-hypeman-best-shine-at-soundcity-on-tour/)则说明 DJ + hypeman 组合仍活跃于大学生受众。

这组证据意味着尼日利亚采集应建两列：

1. MC/host 的 rules、games、run of show、crowd control；
2. hypeman 的 lamba、chants、call-and-response、DJ transitions、viral sound。

### 最像“局头备忘录/手册”的短视频形态

[Travel Noire 对 Lagos Detty December 的报道](https://travelnoire.com/detty-december-lagos/)记录了一位创作者做 TikTok “rules”系列，教人如何在 Lagos 的 Detty December 安全、聪明地玩。它不是卡牌游戏教程，但形式上非常接近：

- 规则 1、规则 2 连载；
- 场景高度明确；
- 创作者拥有“懂局的人”人设；
- 内容是可保存、可转发的派对备忘录；
- 规则同时覆盖社交、预算、出行和人身安全。

这是五国里目前最接近“局头手册内容流”的公开可复核海外样本。

### 本地 Truth or Drink 变体

- 内容词：real gist；

- 五秒内回答，否则喝；
- 全组投票判断是否撒谎，可能翻倍；

其中“场景分层、双语规则卡、由浅入深题库、无酒精替代”值得吸收；“victim、五秒强迫、全组判谎、翻倍喝、掌掴或强制运动”必须拒绝。

[NotJustEvent 的尼日利亚派对游戏文章](https://blog.notjustevent.com/10-unique-party-game-ideas-for-hosts-planers/)又把 host、planner、MC、crowd、vibe、owambe、abeg、party sweet die 放进同一语境，适合扩展本地口语查询。

## 风险语料：可以搜，但不能照搬

以下词和机制在源内容中真实存在，但应进入过滤器，而不是创意库默认项：

- 越南：nặng đô、喝两三杯、翻倍、拒绝就罚；
- 泰国：หมดแก้ว、ดื่มรอบวง、สายแข็ง；
- 波兰：不做/不会就喝、强制 kara；
- 尼日利亚：victim、五秒强答、全组判谎、double drink、掌掴、强制俯卧撑；
- 所有地区：成人亲密问题、暴露隐私、强迫触摸、替别人打电话、发布朋友照片、公开羞辱。

安全改写底线：

1. 所有饮酒结果必须能一键替换成无酒精饮品、记分、换题或休息；
2. pass 不需要解释，不设置二次惩罚；
3. 不让全组裁决一个人的真诚、性经历、关系忠诚或身份；
4. 不强迫身体接触、公开通话、发帖或上传朋友照片；
5. 不设计喝完、连续喝、翻倍喝和速度竞饮；
6. 成人题库单独开关，默认关闭，并提示 consent；
7. hypeman/黑话内容不得把财富、性别、族群、阶层或非法赚钱方式当作羞辱或炫耀素材。

## 推荐的下一轮站内验证方法

如果后续能稳定使用各平台已登录状态，应按“词根—内容—评论—创作者网络”四层采，而不是只搜一次关键词。

### 第 1 层：精确词根

每个本地词至少组合三种语境：

- 角色词 + party/game；
- 角色词 + rules/tips/notes；
- 角色词 + local occasion。

例如：

- Mistrz Gry + domówka / zasady / wyzwania；
- hypeman + Detty December / party rules / game night；
- chủ xị + trò chơi / bàn nhậu / luật chơi；
- pencair suasana + game nongkrong / tongkrongan；
- เกมวงเหล้า + คนเปิดกฎ / กติกา / ภารกิจ。

### 第 2 层：标题模板

重点找：

- “规则 1 / Rules I wish I knew”；
- “host notes / game master mode”；
- “things only the organizer knows”；
- “before the party / beforek / bifor / pre-game”；
- “how to keep the vibe / pecah / hype the crowd”；
- “save this for your next house party”。

标题模板往往比翻译“局头备忘录”更容易命中同类内容。

### 第 3 层：评论区变体

评论区需要单独提取：

- 玩家如何称呼负责开局的人；
- “我们家/我们学校/我们圈子是这样玩”的规则变体；
- 两人、小团体、宿舍、婚礼 after-party 的改版；
- 安全替代：water、non-alcohol、pass、skip、no-go；
- 风险升级：finish、double、victim、hardcore、nặng đô、สายแข็ง。

### 第 4 层：角色创作者和声音网络

- 波兰追 Mistrz Gry / Melo / LocoGames 相似创作者；
- 尼日利亚追 hypeman + DJ + viral sound，而不只追“party game”；
- 越南追 bài Huệ / NocOut / U Là Trời 的买家内容和规则演示；
- 泰国追 เกมวงเหล้า 的发规则、做轮盘、做卡牌内容账号；
- 印尼追 nongkrong、pencair suasana、impostor 和传手机玩法账号。
