## 结论

海外平台没有出现一个能逐字对应“局头备忘录”的固定角色词。真正稳定的是一组功能等价、但按场合和内容格式组织的词：`pregame` / `predrink` 描述出门前的聚会场景，`game night host ideas` 描述主持与筹备，`drinking game prompts` 和自制卡组承载提示词清单，而 `drunk spelling bee`、`Stir the Pot`、`Calabria game` 等是可被反复复刻的命名玩法。

因此，海外检索不能只翻译“局头、手册、笔记”。更有效的路径是从场合词进入，再沿平台建议、标题变体、命名玩法和自制提示卡向外扩展。

## 海外真实词河

```text
pregame / predrink / pres
  ├─ pregame games / game night ideas
  │   ├─ how to host a game night
  │   ├─ drinking game prompts / homemade card deck
  │   └─ party-game non-negotiables / budget / food
  ├─ drunk spelling bee
  │   ├─ drunken bee / tipsy spelling bee
  │   ├─ Tequila & The Bee
  │   └─ Spell or Shots
  └─ named games
      ├─ Calabria game / I love London
      ├─ Stir the Pot
      ├─ You Lie You Drink
      └─ Golf Pong
```

这是一组发现路径，不是同义词表。`game night host`更接近主持流程，`drinking game prompts`更接近备忘录内容格式，`pregame`是场合，命名游戏则是机制载体；它们不能被合并成一个“海外局头”的虚假翻译。

## 安全边界

- 含 `drunk`、`drink`、`shots` 的命名只作为搜索证据；产品化时默认改成水、无酒精饮料、积分或跳过，不把饮量和速度作为胜负条件。
- `Stir the Pot` 的“朋友互评/roast”可能引发羞辱、排挤和关系伤害；只保留自愿、正向、可跳过的提问框架。
- 手写提示卡中出现性暗示、裸照、隐私账号和身份特征时，只记录“存在风险类别”，不复制成可直接执行的题库。
- 任何拼字、舞蹈或卡牌玩法都应允许清醒参与者和不饮酒参与者取得同等地位。
