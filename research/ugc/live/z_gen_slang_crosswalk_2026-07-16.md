# Z 世代组局/派对黑话与海外功能对照（2026-07-16）

## 先说最重要的

这不是翻译表。“局头”在中文平台同时包含召集、订台、带气氛、带游戏和商业招募；海外公开样本更常把这些功能拆成 `pregame` 场合、`game night host` 主持、`drinking game prompts` 提示卡和若干命名玩法。强行找一个英文单词对应“局头”，会漏掉大部分真正内容。

## 中文平台词族

| 层 | 已观察词 | 本轮判定 | 采集价值 |
| --- | --- | --- | --- |
| 角色 | `局头`、`通天代/通天带` | 强；后者的双关解释仍需复核 | 反查作者、系列和带台内容 |
| 相邻角色 | `派对师`、`酒吧氛围组` | 活跃但歧义 | 只能搭配酒吧、抓手指、带台等锚点 |
| 内容载体 | `局头备忘录`、`带台备忘录`、`局头笔记版`、`局头必备攻略`、`局头网课` | 强 | 最适合抽机制、标题模板和系列 |
| 教程动作 | `带台`、`前摇`、`抓手指`、`对抓对指` | 前三项强；最后一项混有噪声 | 找主持动作、暖场、反应与节奏机制 |
| 待核动作 | `逆抓顺打`、`中抓`、`逢三抓手` | 候选 | 需要至少 20 条结果和多作者复核 |
| 联络招募 | `组局dd`、`酒搭子dd`、`同城组局`、`今晚组局搭子` | 强但非玩法 | 找场景与作者，不直接入玩法库 |
| 外貌门槛 | `颜局`、`卡颜局`、`微卡颜` | 强词族、风险高 | 只进风险词典，不做排斥规则 |
| 价格/资格 | `女A男免`、`男A女免`、`通免` | 强词族、风险高 | 记录市场语言；禁止性别差别推荐 |
| 评价模板 | `从夯到拉`、`锐评`、`顶帅打法` | 前两项强，后一项候选 | 找榜单/评论内容，避免评价真人 |

`dd`、`A`、`通天代/带`的具体来源和词义含有语境推断。仓库记录的是平台中反复出现的搭配，不把推断伪装成正式词典定义。

## 海外平台词族

| 层 | 已观察词 | 本轮判定 | 对应功能 |
| --- | --- | --- | --- |
| 场合 | `pregame`、`predrink`；英国分支 `pres/pre-drinks` | 强 | 出门前或正式活动前的朋友聚会 |
| 主持/筹备 | `game night host ideas`、`how to host a game night` | 强 | 更接近“怎么带局/怎么准备” |
| 提示清单 | `drinking game prompts`、`homemade card game/deck` | 强 | 最接近“备忘录/笔记”的内容载体 |
| 命名玩法 | `drunk spelling bee`、`Tequila & The Bee`、`Spell or Shots` | 强 | 可复刻的主持型比赛模板 |
| 朋友互评 | `Stir the Pot`、`You Lie You Drink` | 强但高风险 | 点名/判断/互评；要防羞辱和关系伤害 |
| 舞蹈/节奏 | `Calabria game`、`I love London` | 强单源、名称有地方变体 | 通过音乐/动作组织群体节奏 |
| 宽泛探针 | `drinking game host`、`party game prompt cards` | 弱或歧义 | 用来长出新词，不作为最终标签 |

## 功能对照，不是逐字翻译

| 中文需求 | 海外更有效的检索入口 |
| --- | --- |
| 找“局头怎么带” | `game night host ideas`、`how to host a game night` |
| 找“局头备忘录/笔记” | `drinking game prompts`、`homemade party game cards` |
| 找“局头网课/系列” | 命名玩法 + `how to host` + 创作者主页反查 |
| 找“抓手指/前摇”同类 | 从 `pregame games` 进入，追踪重复出现的命名玩法和动作游戏 |
| 找“组局”场合 | 美国 `pregame/kickback`；英国 `pres/pre-drinks`；再搭配 games/host |

## AI 单独猜不到时怎么找

1. 用一个粗角色或场合词启动，例如 `局头`、`pregame`。
2. 原样记录平台自动补全、“大家都在搜”、相关搜索和筛选标签。
3. 把新组合逐个回搜，不做同义改写。
4. 检查是否有精确标题、多位作者、系列编号或高频创作者自称。
5. 从创作者后缀、标题模板、评论中的别名继续长词。
6. 把词分为角色、场合、内容载体、动作、评价、联络和风险七层，避免把所有词都误当玩法。
7. 只有多作者重复、前排结果密度和固定内容形态同时成立，才升级为正式种子。

本轮最典型的发现链是：

```text
局头
→ 局头备忘录 / 局头带台游戏
→ 局头网课 / 抓手指前摇
→ 局头必备攻略 / 通天代(带)
→ 从夯到拉 / 锐评 / 投稿小程序
```

海外对应的真实词河则是：

```text
pregame
→ pregame games / game night host ideas
→ drinking game prompts / homemade deck
→ drunk spelling bee / Stir the Pot / Calabria game
→ 标题变体、评论别名和创作者系列
```

这套方法的核心，是让平台行为产生候选，让 AI 只负责去重、分类、反查和风险标注。
