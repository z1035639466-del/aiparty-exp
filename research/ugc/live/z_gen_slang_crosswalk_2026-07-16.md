# Z 世代组局/派对黑话与海外功能对照（2026-07-16）

## 先说最重要的

这不是翻译表。“局头”在中文平台同时包含召集、订台、带气氛、带游戏和商业招募；海外公开样本更常把这些功能拆成 `pregame` 场合、`game night host` 主持、`drinking game prompts` 提示卡和若干命名玩法。强行找一个英文单词对应“局头”，会漏掉大部分真正内容。

### 第二波校正：哪些词真是玩法，哪些只是夜场市场话语

第二轮站内回搜进一步排除了几组“听起来像圈内玩法、实际不是”的词：

| 原样查询 | 可见样本 | 判定 |
| --- | ---: | --- |
| `逢三抓手` | 18 条，至少 12 条相关 | 已验证的反应/破冰玩法名；横跨培训、团建、酒桌与带台 |
| `逆抓顺打` | 至少 20 条，相关 0 | 几乎全是麻将/牌类，不并入聚会玩法词河 |
| `逆抓顺打 局头` | 19 条，精确相关 0 | 只回退到普通局头与抓手指结果 |
| `什么叫中抓` | 15 条，相关 0 | 几乎全是音声、广播剧语境 |
| `收枪版` | 19 条，相关 0 | 主要是游戏皮肤/武器交易 |
| `抓手指收枪版` | 18 条，精确相关 0 | 能回退到抓手指内容，但不能证明“收枪版”是稳定变体名 |

`酒吧A仔` 则是强命中，但它属于**夜场角色/招募/价格风险词河**，不是玩法：13 条可见卡片几乎全部相关。两条固定科普帖共同支持 `局头—枪手/代理—A仔—男免/女免—营销` 这类分层；其中 `A仔` 多指普通付费或分摊参与者，而非主持人。`罗汉局 / 女围局 / 皇帝局 / 卡颜局 / 洗客` 等也应进入风险词典，不能拿去生成排斥、性别差别定价或招募规则。

评论里出现的 `女甩`、`👋6`、`卡代理`、`得吃` 等只有单条或少量解释，先保留为低置信度回搜种子，不伪装成统一定义。完整证据见 [小红书第二波](./xhs_second_wave_2026-07-16.md)。

抖音在验证码解除后的章节级证据又补出一个更准确的角色模型：`营销` 偏订台和接待，`局头` 位于营销与玩家之间并承担招募、跨场活动和带台功能，熟练玩家也可能因为会带台而被误认成局头或枪手。因此它不是稳定职位三分法，而是**商业功能 × 带台能力 × 玩家身份**的连续谱。

另一条“12 种局”把拥挤局、`罗汉局`、`卡颜局`、性别比例高度偏斜的局、后场模特局、`通A/AA` 拼卡、`女A男免 / 男A女免` 和朋友局放在同一套夜场 taxonomy 中。它进一步证明这些词主要描述人群、付费、招募和门槛，不是可执行玩法。完整证据见 [抖音夜场黑话原生补扫](./douyin_nightlife_slang_live_2026-07-16.md)。

`dd`、`A`、`通天代/带`的具体来源和词义含有语境推断。仓库记录的是平台中反复出现的搭配，不把推断伪装成正式词典定义。

## 功能对照，不是逐字翻译

| 中文需求 | 海外更有效的检索入口 |
| --- | --- |
| 找“局头怎么带” | `game night host ideas`、`how to host a game night` |
| 找“局头备忘录/笔记” | `drinking game prompts`、`homemade party game cards` |
| 找“局头网课/系列” | 命名玩法 + `how to host` + 创作者主页反查 |
| 找“抓手指/前摇”同类 | 从 `pregame games` 进入，追踪重复出现的命名玩法和动作游戏 |
| 找“组局”场合 | 美国 `pregame/kickback`；英国 `pres/pre-drinks`；再搭配 games/host |

## AI 单独猜不到时怎么找

1. 用一个粗角色或场合词启动，例如 `局头`、`pregame`。
2. 原样记录平台自动补全、“大家都在搜”、相关搜索和筛选标签。
3. 把新组合逐个回搜，不做同义改写。
4. 检查是否有精确标题、多位作者、系列编号或高频创作者自称。
5. 从创作者后缀、标题模板、评论中的别名继续长词。
6. 把词分为角色、场合、内容载体、动作、评价、联络和风险七层，避免把所有词都误当玩法。
7. 只有多作者重复、前排结果密度和固定内容形态同时成立，才升级为正式种子。

本轮最典型的发现链是：

```text
局头
→ 局头备忘录 / 局头带台游戏
→ 局头网课 / 抓手指前摇
→ 局头必备攻略 / 通天代(带)
→ 从夯到拉 / 锐评 / 投稿小程序
```

海外对应的真实词河则是：

```text
pregame
→ pregame games / game night host ideas
→ drinking game prompts / homemade deck
→ drunk spelling bee / Stir the Pot / Calabria game
→ 标题变体、评论别名和创作者系列
```

这套方法的核心，是让平台行为产生候选，让 AI 只负责去重、分类、反查和风险标注。
