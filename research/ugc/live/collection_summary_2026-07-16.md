## 全球青年黑话河口扩展

在原有多语玩法采集之上，又按“精确搜索后是否持续长出玩法、主持技巧、规则、卡牌、脚本、App 或系列创作者”完成全球词河扩展：东亚/东南亚 36 条、欧洲/拉美 46 条、英语圈/印度/尼日利亚/南非 21 条，共 103 个词簇（50 verified、34 candidate、12 dead/negative、7 risk-only），并另做 18 条 YouTube 原样回搜与平台冲突审计。

本阶段最重要的校正是：**词义像“局头”不等于能当采集入口。** 当前搜索行为最接近的单角色词是越南语 `quản trò`；韩国 `술게임 장인` 是值得继续判活的青年角色候选；日本 `幹事必見 + 飲み会ゲーム` 与印度 `freshers party anchoring script` 更像“局头手册”的内容壳。大多数市场的主矿脉则是本地场景黑话与内容载体组合，如 `la previa → juegos para la previa`、`esquenta → Esquenta, O Jogo`、`pre-drinks → games`、`tongkrongan → permainan/tebak-tebakan`。

## 对原问题的直接答案

`局头备忘录`是真实、强势的小红书/抖音内容格式；`局头笔记`有真实结果但密度较弱；`局头手册`本身接近死词，平台会把它改写回备忘录、笔记和教程。

最有效的发现方法不是让 AI 猜近义词，而是：

1. 从粗角色词 `局头` 启动；
2. 抄录自动补全、相关搜索和标题共现；
3. 原样回搜 `局头备忘录`、`局头带台游戏`、`局头网课`；
4. 从精确标题和创作者自称继续长出 `抓手指前摇`、`局头必备攻略`；
5. 再跨到抖音，验证 `通天代/通天带`、`从夯到拉`、`锐评` 和投稿型玩法小程序；
6. 同时建立风险词河：`组局dd`、`卡颜局`、`女A男免`不能混进玩法库。

这些词很多来自游戏和短视频话语的迁移：`前摇`、`通天代`、`打法`、`排位`、`单排`被拿来描述带台动作、能力和排名。它们是平台共现发现，不是普通词典翻译。

第二波又得到两类关键校正：

- `逢三抓手` 在 18 条可见样本里至少 12 条与培训破冰、团建、酒桌或带台相关，已经从候选升级为真实玩法名；
- `逆抓顺打 / 什么叫中抓 / 收枪版` 在对应查询里被麻将、音声或游戏武器交易占满，不能因为词形像圈内话就硬塞进玩法库；
- `酒吧A仔` 的 13 条前排卡片几乎全部相关，但它属于夜场角色、招募和价格风险语境。`A仔 / 枪手 / 代理 / 男免女免 / 卡颜局` 等只进风险词典，不进入玩法生成器；单评论解释的 `女甩 / 👋6` 仍保持低置信度。

## 海外同构内容

海外没有发现一个稳定、跨文化、能逐字等于“局头”的统一黑话。功能被拆成三层：

- 场合：`pregame / predrink / pres / previa / preserata / présoirée / esquenta / Vorglühen / 술자리 / 飲みゲー / inuman`；
- 控流程的人或载体：`game master`、具体玩法的局内角色、game-night host、单手机主持、App、题卡和规则库；
- 命名内容：`drunk spelling bee / Tequila & The Bee / Spell or Shots`、`Stir the Pot`、`Calabria game / I love London`、自制 prompt deck。

因此，海外“局头备忘录”的真实检索式更接近：

```text
本地场合黑话 × host/game master × prompts/cards/rules/tutorial × 平台形态
```

例如 `pregame games` 会长出命名玩法，`drinking game prompts`会长出手写卡组，`game night host ideas`会长出主持流程；三者合起来才是功能同构。

本轮多语登录态进一步证明本地场合词的重要性：西语 `previa`、法语 `apéro`、越南语 `nhậu` 都比逐字翻译更能长出原生内容；意大利样本把本地电视节目改成聚会格式，巴葡样本把玩法嵌入赛事间歇和 BGM，德国样本则暴露电子游戏失败叠加烈酒的高风险路径。

韩国最强的“手册同构”是 `술게임 강의 / 특강 / 01강 / 02강 / 10초 술게임` 这组课程与短演示载体。`인싸 / 핵인싸 / 분위기 메이커` 更像气氛身份，不应自动等同于负责讲规则的人。

- 抖音“夜店酒吧黑话”检索最初触发拖拽验证；用户完成验证后已重跑成功，前排 18/18 与局头、组局、A仔、卡颜、带台或行业科普相关，并核验了三个固定详情页。抖音仍不是全站逐视频/逐评论穷尽采集。
- Facebook 菲律宾已用登录态完成两条 Reels 原生查询并核验两个详情页；规则信息不足的页面没有补猜或入原子。私人群、群内评论和成员信息没有采集，也不会用加入群组来扩大样本。
- Pinterest 印度已完成两条登录态首屏查询，21 张 Pin 中保守 14 张相关并固定 5 个 Pin 页面；图片题表和答案没有复制，board 级去重与更大规模采样仍是空白。
- TikTok 已完成英语、西、意、法、巴葡、德、越以及泰、印尼、波兰、尼日利亚的登录态查询与详情抽样。后四地在用户明确让出 ADS 操作后补跑，共检查 32 张首屏卡、30 张相关；它仍是每地一轮发现快照，不是全部视频/评论的全量导出。
- 韩国 YouTube 登录态完成 10/10 相关首屏和一个详情页；另有 6 条公开视频确认自动韩语字幕轨存在，但逐段字幕正文没有成功取回，未伪装为已转写。
- 数 GB 至数 TB 的 dump/torrent 只登记体量、授权和选择性抽取路径，没有无差别下载。所谓“尽可能全”落实为完整入口地图与可继续运行的采集器，而不是把未经筛选的原始互联网全部落盘。

`p0_query_log.csv` 的 `verified` 表示本轮页面快照里有直接、重复或机制级证据，不等于每个词都达到了“自动补全 + 至少 20 条人工样本”的正式批量种子晋级门槛。样本不足 20、只核验单帖或缺自动补全的词仍须保留日期和范围限定，不能对外声称全平台流行度。

## 安全结论

年轻人内容里的“黑化”更多表现为成人模式、羞辱/锐评、强饮、外貌门槛、性别差别定价、强配对和默认录制。报告保留这些词用于识别与过滤，不把它们整理成可直接执行的危险玩法。所有产品化建议默认：无酒精替代、可跳过、显式同意、不公开真人排名、不索取私人设备和照片、不做金钱或性别门槛。
