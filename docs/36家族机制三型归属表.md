# 36 母机制家族 → 机制三型归属表（三问机械分类）

> 存量欠账清点之一。把研究侧归并出的 **36 母机制家族**按 **DM-skill v1.9「机制三型」骨架**做一次机械归属，边界件单列待人工复核。
> 来源：`research/01_线下游戏采集.md`（家族 1–20）、`02_重点八国补采.md`（21–26）、`03_第二轮补采…md`（27–36）；三型与死约束原文见 `docs/specs/DM-skill-v1.9.md` §适配参考、`stage1-final-routing-v1.md` §v1.9 范围第 2 项。
> 归属只看**胜负判定的底层结构**，不照抄家族的表层主题或名字。

## 机制三型（每家族归属其一）

- **候选池型**：胜负要求从**在场玩家**中锁定一个**跨轮持久**的谜底（隐藏身份/阵营推理、持久藏物）。死约束 **N≤5 禁作核心循环**（候选池 ≤4 时信息坍缩与信息真空之间无可玩区间）。
- **对抗型**：玩家间相互加码/押注/竞速对抗驱动（叫价诈唬、递增博弈、竞速对抗）。死约束**递增/加注必须到顶强制摊牌**；N≥6 配旁观押注/分组防串行等待爆炸。
- **广播型**：各自对同一刺激独立反应、独立计分（自曝陈述、表演猜词、接龙、传递链、轮盘、限时挑战——绝大多数家族）。N≥3 无约束，**小局首选**。

## 三问机械分类（有序决策树）

1. **Q1**：胜负是否要求从在场玩家中锁定一个跨轮持久的谜底（隐藏身份/阵营/持久藏物）? → 是 → **候选池型**。
2. **Q2**（Q1=否）：是否由玩家间相互加码/押注/竞速对抗（递增博弈、叫价诈唬）驱动? → 是 → **对抗型**。
3. **Q3**（Q1、Q2=否）：各自对同一刺激独立反应、独立计分? → 是 → **广播型**（默认归属）。

> 遇边界件（三问树无法干净判定，或名字诱导误触 Q1）**不强归**，落「§边界件·待人工复核」。

## 归属统计（不含 4 个边界件）

**候选池型 1**（家族 8）｜ **对抗型 6**（10、12、17、21、22、28）｜ **广播型 25**（其余）｜ **边界·待人工复核 4**（3、16、33、34）——合计 36。

> 印证原框架：广播型为"绝大多数家族"的默认归属；候选池型仅隐藏角色社交推理一族为纯原型；对抗型集中于叫价诈唬 / 竞速抢物 / 爬梯甩牌 / 叫牌吃墩 / 竞速夺位棋等"递增·竞速·加码"驱动家族。**分队**作正交修饰符独立于三型，与四个待复核边界件一并留人工终判。
