推荐方案 · B 为本体，C 作分身 一个灵魂 · 两副面孔 · 一把嗓子
结论：B「有魂光球」当本体——它是唯一一个在极光皮里不出戏、又真的「有个东西在场」的方案。 进入糖果皮（战报/分享/贴纸）时，同一双眼睛落到 C 的骰子头上——玩家立刻认出「还是它」。 D 的文字人格不作为独立方案，而是这颗球的嗓子：眼睛管情绪，嘴管性格。A 保留为球的「深呼吸」状态（生成中无面部）。
