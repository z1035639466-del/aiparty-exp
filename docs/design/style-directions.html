给一桌人，现制一局
为今晚量身的派对游戏。

aiparty 是线下派对游戏的「局长」：读一桌人的口味（偏好滑块）和素材（他们给的梗）， 当场编出只属于这群人的一局。四版视觉都按大厂产品级的做工来——Linear 的精工、Arc 的表现力、Stripe 的克制—— 各占一种气质，同一套机制词汇，皮不同、骨相通。

怎么选 · 一句话对照

A 极光

要「潮 + 高级」。C 端旗舰。风险：玻璃拟态要克制，别糊。

B 石墨

要「精准可信」。硬核 / B 端 / 引擎。风险：派对暖度偏冷。

C 白日

要「亲和零门槛」。破圈。风险：个性偏淡，需靠内容出彩。

D 糖果

要「传播上头」。战报分享。风险：长时间看会累，宜做点缀。

推荐组合：主视觉走 A 极光——最贴「为我们现制一局」的惊喜内核，又是四版里最耐看的高级感； 结算/分享环节切到 D 糖果 制造传播爆点；B 石墨 沉淀为后台与生成引擎的一致语言。三皮一骨，共用同一套机制词汇。
