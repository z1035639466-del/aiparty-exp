过渡处怎么接？
对局最后一票落定 → 屏幕从极光暗底「翻面」成糖果撞色（0.4s 卡片翻转），机制词汇不变： 身份牌、声明质疑、连蒙对 在两张皮里是同一套 settlement 数据，只是换了张脸出场。

分享卡回流入口 = 拉新钩子：好友点开战报 → 落地 极光 配局页，动线闭环。
