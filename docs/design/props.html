aiparty · 道具设计 v2 · 实物还原

桌上的东西，归现实。

推翻上一版的「道具穿品牌皮」思路。道具是玩家在桌上摸过千百次的实物——KTV 的骰盅、麻将桌上的骰子—— 数字版长得越像真的，上手越快、越可信。界面壳归软件（极光），桌面上的道具归现实。

① 现实有的 → 数字孪生

材质、比例、物理、声音照抄现实：骰盅是哑光黑筒红绒口，骰子是白亚克力、幺四红点、凹坑点数。不加光效不换色。

② 现实没有的 → 才自己设计

局长本体、身份牌背面纹样、密语卡这类现实无原型的，才轮到我们的视觉语言出手。

③ 壳是软件，桌面是实物

按钮、标签、倒计时等 UI 浮层保持极光语言（半透明浮在桌面上方）；桌布、道具永远拟真，两层不互相污染。

关于左轮枪形 · 上一版为什么去掉，这一版为什么恢复

上一版「只留弹巢」是我做的保守判断——顾虑应用商店年龄分级、家庭场合观感与内容审核，不是硬性规定。 重新评估：派对游戏语境里道具指向明确，狼人杀/剧本杀类产品中枪形道具大量存在且正常过审。 本版恢复完整左轮为默认形态；「中弹」时刻从枪口弹出一面「罚」字小旗——这本身就是现实里存在的整蛊道具（bang flag gun）， 既保住真枪质感，又让「开枪=开个玩笑」不言自明。枪形即唯一形态——真遇到分级问题，到时再改就是。

这一版的分界线很清楚：可信感来自现实，惊喜感来自软件。桌上三件套一比一还原实物（连幺四红点、绒口骰盅、棘轮咔哒都照抄）， 软件只做现实做不到的三件事——把盅内只给你一个人看、把随机做成谁也赖不掉的公正、把每次结果记进账本。
