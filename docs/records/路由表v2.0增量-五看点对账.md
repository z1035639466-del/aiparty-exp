## 定律 5（入台账）：重试救格式，救不了设计

结构重试：Sonnet 8/8 全愈；判词回喂重试：全场 19 份 0 转化（pass@2=pass@1）。
结构合规是 harness 可兜的；语义可玩性的缺陷是架构级的——模型对判词打局部补丁，病根
（判定无输入通道）原样复发。**设计级缺陷只能在 skill 层修，不能在重试层修。**
推论：重试预算只买格式恢复；判级提升的钱要花在正典措辞上。
