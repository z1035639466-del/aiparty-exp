# 裁定 · 产品定位:线下社交游戏局(北极星句)

> 2026-07-22 房主裁定补落。此前文档只有架构定位(Mode A/B),产品定位从未成文,
> 导致全链路向文本坍缩(三局 episode 道具工具调用≈0,病历见 61 卡体检与道具箱铁律)。

## 北极星

**ZAKZOK(原代号 Yappa)是线下社交游戏局的活局长,不是聊天局,不是反应手感游戏 App。**

游戏发生在桌上、身体上、道具上、人和人的眼神里——不发生在屏幕里。

## 目的(2026-07-23 房主补落:宪法的第二条腿)

**局的目的是促进关系:破冰、靠近、让同处一室的人玩完之后关系更近。**
整蛊、藏信息、认罚、对峙全是手段——制造共同经历和情绪波动,让关系升温。
宪法自此两条腿站立:**不靠文字(场地)+ 为了关系(目的)**——前者防向文字坍缩,
后者防向纯整人坍缩(只剩尴尬和输赢的局,赢了也是输)。

## 手机的四个角色(仅此四个)

1. **入座票**:进桌、看进度、三信号。
2. **私密信道**:藏信息的唯一载体——私件、私骰、技能牌、额头牌。
3. **判定仪器**:显式判定时刻才抬起来——快枪手拔枪、拍照判定、录音判定、场景侦察。
4. **局长的喇叭**:台词与音乐。

## 判据(设计决策拿这个拦漂移)

- 一个新功能如果让玩家**盯屏幕的时间变长**而不是让桌上的人**互相动起来**,默认拒。
- 一拍下来桌上没人动身体/动道具,是失败拍(局长侧已入【道具箱】铁律)。
- 屏上游戏时刻(duel 类)是**判定例外**,每局限量,不是产品主体。
- 文字是组织手段,不是游戏场地。
- **道具只发不代玩**:局长把道具/玩法发下去,结果由玩家在现实里**自己玩出来**,再经问询(ask)或
  玩家主动上报(say to=局长)入账。局长替玩家玩了再把结果告知(替玩家摇骰/抛硬币/翻牌),
  与虚构玩家发言**同罪**——玩的动作必须落在玩家手指上,不是多盯一块屏。骰盅道具(prop.dice_cup +
  玩家端摇一摇)是这条的第一个落地件:局长发盅,玩家自己摇。
- **为关系服务**:一个功能/玩法如果只制造尴尬或输赢、不产出"玩完之后关系更近了",是失败的。
  情绪波动要落在"共同经历"上,不落在个人难堪上(与结算底线"负向标签禁令"同源)。
- **素材气质随目的走**:对外素材做"一桌本来不熟的人玩完笑作一团",不做"看谁被坑惨了"——
  前者受众是聚会/团建/约会/新朋友局的所有人,后者只剩爱起哄的那拨。整蛊是引子,关系是产物。

## 判据补(2026-07-22 追加:道具技术选型五步——防"向像素坍缩")

第二次坍缩病历:骰盅视觉从"预渲染还是实时 3D"吵起,Blender 物理/Unity 调研/RN 假 3D
全在解一个不存在的问题——**摇骰子是盖着摇的,摇的那几秒屏幕本来就没有像素**;
揭盅那一眼是静态的。媒介自带引力(渲染诱人),无宪法则必坍缩,与文本坍缩同构。
每次道具决策按此顺序走:

1. **先过宪法再选技术**:这功能让人盯屏↑还是让桌上的人动起来↑?前者默认拒。
2. **把道具拆成"时刻",逐个问注意力在哪个感官通道**(例:摇=手/震动 → 揭=眼/一帧 → 记=脑)。
3. **通道决定技术,不是反过来**:手=震动,眼=一帧静态图或短动画,耳=音效。
4. **基线先做全**:点击+视觉+音效,任何道具仅靠基线就完整可玩;震动/体感是增强层,
   不许先做增强层后补基线(病例:骰盅先上了 Core Haptics,音效缺席,安卓/关触感用户零反馈)。
5. **只有"连续输入→连续输出"的时刻才值得碰渲染/引擎**——道具库 14 件全是离散事件
   (摇/扣/翻/传),无此时刻,**引擎线(Unity/GL/物理渲染)永久关闭**,不再复议。

## 病历(为什么要成文)

工具面/prompt/App 全部以文本为默认输出面,LLM 引力叠加,无宪法则必坍缩。
症状级补丁已打:【道具箱】(嘴摇骰=虚构)、【循环开】、random.dice 实装、骰面渲染;
骰盅道具(局长发盅、玩家摇一摇自己摇)把"道具只发不代玩"从主持词提到了产品结构。
第二次坍缩(向像素)的复盘产出上方五步判据,由道具官(本地 Opus)复盘、房主裁定入宪。
**第三次坍缩(向冲突,2026-07-23)**:agent 两次把产品框成刺激侧("AI 毒舌""整到破防"),
把"关系/温度"丢掉——冲突比温情抓眼球,LLM 与素材都天然往刺激跑,与前两次同构。
疫苗即上方【目的】段:目的句在,读到的人不会自由落体到"整人"。
三次坍缩同一条规律:**媒介/素材自带引力,宪法缺一句,工作就往那个方向漂**——
所以每发现一种新引力,就补一句宪法,而不是在对话里点头了事。
本文为宪法级,后续任何 prompt/功能/UI 决策先过上面判据。
