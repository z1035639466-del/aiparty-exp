# 美术资产 · 进场时点建议(2026-07-18,Fable 呈房主)

> 原则:美术是**返工成本最高、被上游全方位阻塞**的工序(上游=命名/交互定型/
> 玩法验证)。产品灵魂在内容生成的灵气,美术是放大器不是本体——跟着定稿走,
> 不跟着热情走。AI 生图时代资产周期短,晚开工代价小,提前囤没有红利。

## 时点表(挂里程碑,不挂日期)

| 阶段 | 做什么 | 不做什么 |
|---|---|---|
| **现在(门槛测试期)** | 零人力投入。可选:AI 生图攒 **mood board**(风格方向候选:20 岁感/夜场霓虹/贴纸涂鸦感…),成本≈0,防将来风格讨论从零开始 | 不画任何定稿件 |
| **触发点一:Yappa 律师检索通过** | **品牌最小包**:logo/字形/主色/App 图标/商店占位图——第一笔正经美术投入 | 名字没过检索前一张也不做(报废风险) |
| **触发点二:跨端 App 壳开工 + 模拟台真测把核心交互钉稳** | **UI kit**:玩家端/主持呈现/账本视觉。交互形态(pane 按钮集/挑战展示/仪式呈现)被真人测试定型后才配皮 | 交互还在变时不做 UI 美术 |
| **触发点三:首个 E2 内置件立项**(M-int-1 全量抽取统计出"哪个组件解锁原子最多"后) | **道具资产批**:按件配静态资产(类 UNO 牌组等),风格继承品牌包 | 不按拍脑袋清单囤道具图 |
| **内测出好玩片段后** | 推广物料(AI 视频):素材自产——验收单 S4 传播钩子本来就要求产出画面适合拍 | 不提前拍概念片 |

## 两个现成的免费资产

1. **道具原语扩编批草案的「虚拟态皮」字段**(全屏红光+时钟骤停音效之类)——
   就是未来特效美术的需求单初稿,届时直接转美术 brief,不用重写。
2. **验收单「场面产出」条**:每个好挑战天然产出可拍画面——推广素材的主力
   是玩家拍的真场面,不是画出来的概念图。
