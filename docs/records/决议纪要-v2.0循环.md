## 3. 五看点阈值（终值论证）
**一 1/3+分母≥6**：n=8 的批破因可能仅三四条，小样本占比即掷骰——分母下限双向堵死
（不得触发、不得结案）。**二 2/8+件头注"排期信号非缺陷率"**：量的是文本禁令效力随能力
衰减，用于排期，不入模型缺陷统计。**三 40%/50%/覆盖<6，结案改"本轮未复现"**：原 60/70
太松；覆盖轴堵"分散但贫瘠"盲区（五族各 20% 原阈值全绿）；套皮案定性为常设哨、不可一轮
阴性结案。**四/五 不设触发器+Sonnet 强制解释义务**：零先验凭空拍数会被"跑后不改"锁死；
盯强模型的逻辑=强模型犯错是正典/schema 问题而非模型问题；四五方向相反（合规越高越好/
僭越越低越好）。
