## 2. 通道勾选表（净收益的机判基础，源自语料"标注强在哪"字段）

**增值**：G1 公正/防作弊（杜绝偷看/做局/控向）· G2 隐藏/私密（私发身份、双盲匿名提交）·
G3 记账卸载与留痕（计分/轮次/淘汰名单/回放 verify）· G4 节奏同步（timer/fx 跨设备节拍、
同时揭晓）· G5 护栏（尺度分级、安全提醒、可拒按钮）
**减值**：L1 仪式/眼见为实削弱（Two-Up 型）· L2 掏手机打断身体节奏（抢座位型）·
L3 慢于人眼/流程冗余（逐次共识裁定破坏节奏）
判法：语义净额（非机械减法）——增值≥1 且无显著减值→增值；互抵或皆无→中性；减值主导→减值。
Codex 按表初判 + 人工边界复核。

## 4. 试点（先跑后铺）

- **C 尾组**：投掷精度家族全体 + Pfengeln / Bierdeckel fangen。预注册预期：主体落中性、
  少数增值；开放问题——旁观押注通道计入工具箱后是否翻增值（记录，不预答）。
- **A 段对照组 8 条**：4 条"标注强在哪"型 + 4 条"零损耗/可行性"型。预注册预期：前者
  100% 复现增值；后者部分降中性（健康分化，不算尺失灵）。
- SOP 抽审：4 份 = 2 变动 + 2 维持，含 ≥1 条 A 段对照；任一变动若靠含混通道硬凑即打回。
- 试点过 → 全量并入 04 批；不过 → 修通道清单重试，轴不动。
